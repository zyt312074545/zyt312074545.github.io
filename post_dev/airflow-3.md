# Airflow 实战

对于北漂者，至少对于我来说，每次房租到期，或者由于种种原因需要换房时，每次找房都是一次不小的挑战，不断在各大租房 APP 浏览，，只求安居于一隅，怎奈众里寻他千百度，巨大的信息量不仅将我们湮没于浩浩房源中，无处下手，而且劳心又劳力，非常痛苦，所有想做一个房源信息收集系统，尽量减少在房源中搜寻的时间【本文的数据来源为自如，因为感觉自如的房屋质量与价格勉强成正相关，且没有恶心的后续】。

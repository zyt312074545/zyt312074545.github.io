# 简介

Vimium 是 `Chrome` 的一款插件，可以实现在 `Vim` 的快捷操作。

# 快捷键

```
j         浏览器向下滚动
k         浏览器向上滚动
Shift + j 前一个标签页
Shift + k 后一个标签页
d         浏览器向下滚动一页
u         浏览器向上滚动一页
h         浏览器向左滚动
l         浏览器向右滚动
Shift + h 浏览器向前
Shift + l 浏览器向后
gg        浏览器的最顶端
Shift + g 浏览器的最低端

x         关闭当前标签页
o         进行搜索
Shift + f 标记浏览器可打开位置
b         在当前页面打开标签
Shift + b 在新的页面打开标签

/         在当前页面搜索内容
n         下一个匹配项
Shift + n 上一个匹配项

t         打开新的标签页
Shift + t 跳转到任意标签页

？         帮助文件
```

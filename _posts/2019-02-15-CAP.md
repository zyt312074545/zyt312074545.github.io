# CAP Theorem

对于一个分布式系统，不能同时满足以下三点：

- 一致性（Consistency）
- 可用性（Availability）
- 分区容错性（Partition Tolerance）

# 一致性模型

- 弱一致性
    + 最终一致性
        - DNS
        - Gossip
- 强一致性
    + 同步（主从同步）
    + Paxos
    + Raft
    + Zab
